# 축의금 정리용 프로그램

모바일 환경에서 축의금을 입력하고, 내역과 요약을 확인하기 위한 Flask 웹앱 초안입니다.

## 현재 포함된 기능

- 입력 / 내역 / 요약 3개 화면
- 츄디 느낌의 상단 헤더 + 하단 탭바
- Bootstrap Icons CDN 사용
- SQLite 기본 DB 자동 생성
- 축의금 입력 저장
- 최근 입력 5개 표시
- 내역 목록 표시
- 요약 계산 표시
- 엑셀 다운로드 버튼 자리만 생성

## 설치 예정 패키지

```bash
pip install -r requirements.txt
```

## 실행

```bash
python app.py
```

브라우저에서 접속:

```text
http://127.0.0.1:5000
```

## 기본 입력값

- 구분: 신부측
- 식권 수: 1
- 금액: 빈칸
